Stack Overflow 2017 Survey

Every year, Stack Overflow conducts a massive survey of people on the site, covering all sorts of information like programming languages, salary, code style and various other information. In 2017, they amassed more than 64,000 responses fielded from 213 countries.

This visualization was made to see the technologies people want to used based of the ones that they are currently using.

Data Source

Description

It's purpose

The chart aims to show the technologies that are used by developers around the world and the technologies they want to use based on the ones they are currently using.

How to use

The chart contains a bubble for every technology used by developers. The size of the bubble depends on the number of developers using that specific technology, the more it's used the bigger the bubble. When you hover on a bubble the name of the technology appears with the number of people using it. If you click on one of them, the content of the chart will change and it will contain bubbles of the technologies that developers want to use based on the technology that was clicked.

The list on the right of the chart contains all the technologies that are currently present in the graph with the number of people using a certain technology. If you click on one of them, the content of the chart will change and it will contain bubbles of the technologies that developers want to use based on the technology that was clicked.

Once you've clicked on a bubble (or an item in the list), you can click anywhere in the white area of the chart to return to the main one.
